# Setup und Start

## Voraussetzungen
- .NET SDK 10
- VS Code oder Visual Studio

## Build
```bash
dotnet build ddpc.DartSuite.slnx
```

## Tests
```bash
dotnet test ddpc.DartSuite.slnx
```

## Start API
```bash
dotnet run --project src/ddpc.DartSuite.Api
```

## Start Web
```bash
dotnet run --project src/ddpc.DartSuite.Web
```
