# Chrome Extension: DartSuite Tournaments

## Zweck
Remote-Interaktion mit play.autodarts.io aus der DartSuite heraus.

## MVP-Funktionen
- Lobby öffnen
- Gameplay synchronisieren
- Push-Meldung senden

## Installation
1. `chrome://extensions` öffnen.
2. Entwicklermodus aktivieren.
3. `Entpackte Erweiterung laden`.
4. Ordner `extension/dartsuite-tournaments` auswählen.
