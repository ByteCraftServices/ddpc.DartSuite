# ddpc.DartSuite

DartSuite ist eine modulare Turnier- und Board-Verwaltung für Autodarts.

## Highlights
- Blazor Server Frontend (de/en-US vorbereitet)
- ASP.NET Core API mit Swagger
- EF Core mit InMemory und SQLite
- SignalR Board Status Events
- Chrome Extension MVP
- Unit Tests für Domain/Application/Infrastructure
